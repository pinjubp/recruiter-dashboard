# Recruiter Dashboard

## Structure
- **backend/**: Laravel 12 + Breeze (API mode) + Sanctum + Boost
- **frontend/**: React/Next.js
- **README.md**: Documentation

## Features
- Authentication (Laravel Breeze + Sanctum)
- Resume upload & summarization (AI)
- Job matching & recruiter dashboard

## Setup
1. Clone repo
2. Backend: `composer install`, configure `.env`, run `php artisan migrate`
3. Frontend: `npm install`, run `npm run dev`
